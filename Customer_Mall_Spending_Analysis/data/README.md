# Sample Data File

This directory contains the mall customer dataset.

## Expected File Format

**Filename:** `Mall_Customers.csv`

**Required Columns:**
- CustomerID (Integer)
- Genre (String: Male/Female)
- Age (Integer: 18-70)
- Annual Income (k$) (Integer: 15-137)
- Spending Score (1-100) (Integer: 1-99)

## Sample Data Structure

```csv
CustomerID,Genre,Age,Annual Income (k$),Spending Score (1-100)
1,Male,19,15,39
2,Male,21,15,81
3,Female,20,16,6
...
```

## Data Source
- **Domain:** Retail/Mall Customer Analytics
- **Records:** 200 customers
- **Purpose:** Customer segmentation and spending behavior analysis

## Note
Data files are not tracked in Git for privacy reasons.
