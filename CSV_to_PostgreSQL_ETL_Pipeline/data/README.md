# Sample Data File

This directory should contain your source data files.

## Expected File Format

**Filename:** `patient_health_data.xlsx` (or `.csv`)

**Required Columns:**
- patient_id
- name
- age
- gender
- weight_kg
- height_cm
- bmi
- systolic_bp
- diastolic_bp
- heart_rate
- blood_glucose
- cholesterol
- diagnosis
- medication
- doctor
- hospital
- visit_date
- follow_up_required
- insurance_provider
- notes
- heart_rate_category

## Note
Data files are not tracked in Git for privacy and security reasons.
